# Todo CLI アプリ (日本語)

使い方（基本）:

- 必要パッケージをインストール:

```powershell
python -m pip install -r requirements.txt
```

- 起動:

```powershell
python todo.py
```

コマンド（対話式）:

- `add` : タスクを登録します（タイトル、期限、優先度、場所などを順に入力）
- `list` : タスク一覧を表示します
- `show <id>` : タスク詳細を表示します
- `update <id>` : タスクを更新します（完了にすると自動で削除されます）
- `delete <id>` : タスクを削除します
- `calendar [YYYY-MM]` : 指定月のタスクを日付順に表示します（省略時は今月）
- `help` : ヘルプ表示
- `quit` : 終了

特徴:

- 日付は日本語の自然言語（例: 「明日」「来週の月曜」）で入力可。タイムゾーンは場所に基づき処理。
- 都市名は自動で緯度経度とタイムゾーンに変換（Open-Meteo geocodingを使用、APIキー不要）。
- 天気予報（降水確率・気温）はOpen-Meteoを利用し、期限内で降水確率の低い最適日を候補日として登録。
- 候補日は他タスクと重ならないよう調整。最適日がない場合は空いている日を提案。

注意:

- ネットワークが必要です。
- 依存ライブラリは `requirements.txt` を参照してください。
