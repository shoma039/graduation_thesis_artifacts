# Todo CLI（日本語）

これはローカルで動く対話式のTodoリストCLIです。

主な特徴:
- タスク登録・一覧・詳細・更新・削除
- タスクにID、タイトル、完了フラグ、優先度（1-3）、場所、期限を保持
- 日付入力は日本語の自然言語（例: 明日、来週の月曜）で受け付け、場所のタイムゾーン基準で解釈
- Open-Meteo（APIキー不要）で登録した場所の天気を取得し、期限内で降水確率が低い最適日を候補日として設定
- 候補日は他タスクと重ならないよう調整。候補日が見つからない場合は空き日（予備日）を提案
- タスクを完了に更新すると自動で削除される
- カレンダー表示（月選択で表示）
- 不正入力時に日本語でエラー表示

必要なもの:
- Python 3.9+
- 依存ライブラリのインストール: `pip install -r requirements.txt`

実行:
```powershell
python todo_cli.py
```

備考:
- 都市名から緯度経度はNominatim（OpenStreetMap）を使って取得します（APIキー不要）。
- 天気はOpen-Meteoを使用します（APIキー不要）。
- APIの利用規約やレート制限に注意して下さい。
