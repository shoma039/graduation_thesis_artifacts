# Todo CLI (日本語)

このリポジトリは、ローカルで動く簡易的なTodo CLIアプリです。

主な機能:
- タスクの登録・一覧・詳細表示・更新・削除
- タスクに対して期限を自然言語（日本語）で入力可能（例: 「明日」「来週の月曜」）
- 登録した場所の緯度経度とタイムゾーンを自動で取得
- Open-Meteo を使って期限内の降水確率・気温を取得し、候補日を自動提案
- 候補日は他タスクと被らないように割り当て（被る場合は予備日を提案）
- カレンダー表示（年月指定）

使い方:
1. 依存をインストール
```
pip install -r requirements.txt
```

2. CLIを実行
```
python todo_cli.py
```

注意:
- 外部API (Nominatim、Open-Meteo) を利用します。ネットワーク接続が必要です。
- Nominatim には適切な User-Agent を付与しています。
- タイムゾーン処理には OS の tz データが必要です。Windows環境では `tzdata` を requirements に含めています。
